# Models

Thư mục này dành cho các định nghĩa model / schema (ví dụ dùng với Mongoose, Sequelize, Prisma, v.v.).

Hiện tại demo sử dụng user giả lập trong `services/auth.service.js`. Khi kết nối DB thật, hãy:

- Tạo file model tương ứng (vd: `user.model.js`).
- Import và sử dụng trong các service/controller thay cho dữ liệu giả.
